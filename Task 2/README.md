# Booking System Demo with Max-Heap Data Structure and Heap Sort

This file contains the demonstration for the Resource Booking System with usage of Max-Heap Data Structure and Heap Sort Algorithm.

## Overview

This script demonstrates two concepts applied to the Booking System in Task 1:

1. **Data Structure: Max-Heap (Priority Waitlist)**
    * Staff members are assigned a higher priority (Priority: 2) than Students (Priority: 1).
    * Utilizes a Max-Heap array implementation to handle waitlist.

2. **Algorithm: Heap Sort (Room Sorting)**
    * Utilizes the Heap Sort algorithm to sort the rooms by their capacity in ascending order.
    * Provides an efficient way to display available rooms from smallest to largest.

## Requirements

* Python 3.x

## How to Run the application

1. Download or clone this repository to your local machine.
2. Run the 'heap.py'
